NAME

pcons - Runs a command on the noderange using the out-of-band console.

SYNOPSIS

pcons noderange command

pcons [-h|--help]

pcons [-v|--version]

DESCRIPTION

Runs the a command to the noderange using the console.

EXAMPLES

  pcons 1,3 stat
  pcons all,-129-256 stat

SEE ALSO

psh(1)
