NAME

restartxcatd - Restart the xCAT daemon (xcatd) on AIX.

SYNOPSIS

restartxcatd [[-h|--help] | [-v|--version] | [-r|--reload]] [-V|--verbose]

DESCRIPTION

The restartxcatd command restarts the xCAT daemon (xcatd) on AIX. It uses the startsrc and stopsrc commands to do the restart work. If the xcatd subsystem was not created, restartxcatd will create it automatically.

OPTIONS

-h Display usage message.

-v Command Version.

-r On a Service Node, services will not be restarted.

-V Display the verbose messages.

RETURN VALUE

0 The command completed successfully.

1 An error has occurred.

EXAMPLES

1. To restart the xCAT daemon, enter:

restartxcatd

FILES

/opt/xcat/sbin/restartxcatd
